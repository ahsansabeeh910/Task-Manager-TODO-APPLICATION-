# Task Manager App

<div align="center">


A modern full-stack task management application built using React.js and Express.js.

</div>

---

## Description

Task Manager is a modern full-stack productivity application that helps users create, edit, delete, categorize, and prioritize tasks efficiently. It includes task filtering, priority management, persistent storage, REST API integration, and a responsive UI for improved task organization.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ahsansabeeh910/Task-Manager-TODO-APPLICATION-
cd TODO-APP
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
node server.js
```
